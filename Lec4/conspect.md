## Лекция 4. Условный оператор

### 1. Классический условный оператор
Синтаксис:
```
# Одиночный
if expression:
    body

# С блоком else
if expression:
    body_if
else:
    body_else

# Множественный выбор из разныз условий
if expression1:
    body_1
elif expression2:
    body_2
elif expression3:
    body_3
else:
    body_else
```
